# Database Update Workflow

**Description**: Workflow for database schema changes

## Steps:
1. Analyze required database changes
2. Design new schema or modifications
3. Create migration script with forward and rollback
4. Test migration on development database
5. Update PHP models and classes
6. Update API endpoints if needed
7. Update frontend if needed
8. Test all affected functionality
9. Create database backup before production migration
10. Deploy migration to production
11. Verify production database changes
12. Monitor application performance
